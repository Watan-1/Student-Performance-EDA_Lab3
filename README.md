# Student-Performance-EDA_Lab3

## Course
ART308 - Machine Learning

## Description
This project applies Exploratory Data Analysis (EDA) techniques on the Student Performance dataset.

## EDA Steps Performed
- Checked data structure
- Verified missing values
- Checked duplicates
- Generated visualizations
- Analyzed correlations
- Extracted insights

## Key Insights
- Previous grades (G1, G2) strongly predict final grade (G3).
- Absences slightly reduce performance.
- Gender has minimal impact on final results.
